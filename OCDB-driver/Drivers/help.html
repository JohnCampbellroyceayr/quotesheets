Custom Code Action: ModifyString


Classname: com.zerog.ia.customcode.utils.ModifyString

Description: This class calls allows you to modify a String value. ModifyString makes calls to Jakarta's Regexp, allowing you to search and replace regular expressions based on Regexp 1.2.

Input InstallAnywhere Variables:

$MS_STRING_TO_CONVERT$: Set this variable with the string value that you would like to Modify. This InstallAnywhere variable can be set to the value of another InstallAnywhere variable. (i.e. $MS_STRING_TO_CONVERT$ = $USER_INSTALL_DIR$).

$MS_SEARCH_AND_REPLACE$: Set this variable to "all" or "first" (without quotes).

    all - replace all occurrences of the regular expression ($MS_SEARCH_STRING$).

    first - replace the first occurrence of the regular expression ($MS_SEARCH_STRING$).

$MS_SEARCH_STRING$: Set this variable to the expression you would like to search for.

$MS_REPLACE_STRING$: Set this variable to the string that you would like to replace the $MS_SEARCH_STRING$ expression with.

Output InstallAnywhere Variables:

$MS_NEW_SAR_STRING$: This variable will be set with the new string value based on $MS_SEARCH_STRING$ and $MS_REPLACE_STRING$. (* Remember that $MS_SEARCH_AND_REPLACE$ must be set to true in order for this value to be set)

$MS_PARENT_DIR$: If the value of $MS_STRING_TO_CONVERT$ is a file or directory, this variable will contain $MS_STRING_TO_CONVERT$'s parent directory.

$MS_ROOT_DIR$: If the value of$MS_STRING_TO_CONVERT$ is a file or directory, this variable will contain $MS_STRING_TO_CONVERT$'s root directory.

$ MS_STRING_DBS$: If the value of $MS_STRING_TO_CONVERT$ is a file or directory, this string will contain a converted path containing double back slashes.

$ MS_STRING_FS$: If the value of $MS_STRING_TO_CONVERT$ is a file or directory, this string will contain a converted path containing forward slashes.

$ MS_STRING_LENGTH$: This variable will contain the length of the value of $MS_STRING_TO_CONVERT$.

$MS_STRING_LOWERCASE$: This variable will contain the value of $MS_STRING_TO_CONVERT$ in all lowercase.

$ MS_STRING_TRIM$: This variable will be set with value of $MS_STRING_TO_CONVERT$ with all white spaces removed from both ends of the string.

$ MS_STRING_UPPERCASE$: This variable will contain the value of $MS_STRING_TO_CONVERT$ in all uppercase.

General Instructions:

In order for the action to work correctly make sure to set $MS_STRING_TO_CONVERT$ to the string you would like to modify.

If you would like to make use of the search and replace function, you'll need to make sure to set the following three variables.

$MS_SEARCH_AND_REPLACE$
$MS_SEARCH_STRING$
$MS_REPLACE_STRING$

(* Please see the Input InstallAnywhere Variables section above for more detailed information on how to set these variables)
